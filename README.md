# Gerenciador de Pets

Projeto Vite + API serverless com Swagger UI.

## Variaveis de ambiente

Defina estas variaveis no Vercel (e localmente se for usar `vercel dev`):

- `BLOB_READ_WRITE_TOKEN`
- `JWT_SECRET`

## Rotas importantes

- `/docs` - Swagger UI
- `/openapi` - OpenAPI YAML
- `/autenticacao/login`
- `/autenticacao/refresh`
- `/v1/pets`
- `/v1/tutores`

## Executar localmente

1. `npm install`
2. `npm run dev`
